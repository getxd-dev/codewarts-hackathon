# Bayanihan Bridge PH

Bayanihan Bridge PH is an AI-powered opportunity navigator for marginalized Filipino students, out-of-school youth, fresh graduates, and low-income job seekers.

The MVP turns a short profile plus an uploaded resume, certificate, report card, or handwritten form into an Opportunity Score, matched jobs, free training, scholarships/support programs, skill gaps, and a personalized pathway.

## SDG Alignment

- SDG 4: Quality Education through free courses, training, and education support recommendations.
- SDG 8: Decent Work and Economic Growth through entry-level job matching.
- SDG 10: Reduced Inequalities through barrier-aware support and inclusive scoring.
- SDG 11: Sustainable Cities and Communities through location and access insights.

## Tech Stack

- React + TypeScript
- Vite
- Tailwind CSS
- Recharts
- Local JSON sample data
- Mock OCR / Computer Vision fallback
- Rule-based mock AI recommendation engine
- Deterministic Data Science scoring

## How To Run

```bash

npm run dev
```

Then open the local URL shown by Vite.

## Demo Flow

1. Open the Home page.
2. Click Start Assessment.
3. Fill in a user profile for a student, fresh graduate, out-of-school youth, or job seeker.
4. Upload any resume, certificate, school record, handwritten form, or even a text file.
5. Run OCR. Text files are read directly; other files use demo-safe mock OCR.
6. Generate the pathway.
7. Show the Results page:
   - Opportunity Score
   - readiness classification
   - top 3 jobs
   - top 3 courses
   - top support programs
   - missing skills
   - next 3 steps
   - SDG impact
8. Open Opportunities to show local JSON fallback data.
9. Open Dashboard to show Data Science and SDG impact charts.

## Limitations

- OCR is mocked for images/PDFs unless a text file is uploaded.
- AI recommendations are rule-based and local, not generated by a paid API.
- Opportunity datasets are sample data only.
- No authentication, backend, scraping, or production government integration is included.
